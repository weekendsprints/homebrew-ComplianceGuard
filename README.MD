# ComplianceGuard

Homebrew tap for [ComplianceGuard](https://github.com/weekendsprints/ComplianceGuard) - A cross-platform CLI application for compliance management.

## Installation

### Using Homebrew (macOS and Linux)

1. Tap this repository:
```bash
brew tap weekendsprints/complianceguard
```

2. Install ComplianceGuard:
```bash
brew install complianceguard
```

3. Verify the installation:
```bash
complianceguard --version
```

### One-line Installation

You can also install directly without tapping:
```bash
brew install weekendsprints/complianceguard/complianceguard
```

## Uninstall

To remove ComplianceGuard from your system:

```bash
brew uninstall complianceguard
```

To also remove the tap:
```bash
brew untap weekendsprints/complianceguard
```
